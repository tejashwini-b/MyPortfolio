Wearable Sensor Network System for Health and Safety Applications in Industrial Workplaces

My team and I have published a paper on this project in IJSRCSEIT also we've completed this as final year project We have developed two nodes, namely Health Node and Safe Node. Health node monitors health of the employee by monitoring heart beat rate and temperature of the body. Safe node monitors the external environment by detecting temperature, UV index and other hazardous gases. User is alerted when the threshold limit of the parameters is crossed.
